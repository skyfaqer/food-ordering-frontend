## How to run

First, make sure you have installed Node.js and npm. Then ```git clone``` and enter the project directory.

### Project setup
```
npm install
```

### Compiles and minifies for production
```
npm run build
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Run this on server
```
nohup node prod.server.js &
```
